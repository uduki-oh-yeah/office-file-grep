# office-file-grep

## 概要

Excel / Word / Powerpoint のファイルを検索するためのpowershellスクリプト。

## 対象拡張子

* xlsx
* docx
* pptx

## なんで作ったの？

ソフトウェアのダウンロードが許されない環境で、大量にあるExcelやWordから、特定の文字を検索したかった。

これはpowershellで作っているので、
Windows環境なら大体動くし、
ダウンロードしなくても、コピペでps1ファイルを作れば動かすことができます。

ダウンロードしてないよ！　と言い張れます。
